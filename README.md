# 🏥 InfraOS - Sistema de Chamados Hospitalar
Sistema web simples de abertura e gerenciamento de chamados internos desenvolvido para o Hospital Doutor Ovidio.
O objetivo do sistema é permitir que setores hospitalares solicitem materiais, suporte ou atendimentos internos de forma organizada e prática.

# 🚀 Tecnologias Utilizadas:
- HTML5
- CSS3
- JavaScript

🎯 Funcionalidades:
- Usuário:
- Abrir chamados;
- Informar:
- Nome;
- Setor;
- Nível de urgência;
- Descrição;
- Visualizar chamados cadastrados;
- Status dos Chamados;

🎨 Interface:
- Layout responsivo;
- Design simples e profissional;
- Cores organizadas por prioridade;
- Estrutura pensada para ambiente hospitalar;
